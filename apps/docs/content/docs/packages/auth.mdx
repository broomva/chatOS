# @chatos/auth

Authentication configuration using [Better Auth](https://www.better-auth.com/).

## Providers

- **GitHub OAuth** — `AUTH_GITHUB_ID` + `AUTH_GITHUB_SECRET`
- **Google OAuth** — `AUTH_GOOGLE_ID` + `AUTH_GOOGLE_SECRET`

## Session Management

Better Auth manages sessions via cookies (`better-auth.session_token`). The session cookie is required for authenticated API endpoints like `/api/chat`.

## Configuration

The auth config lives in `packages/auth/` and is consumed by the web app. It uses `@chatos/db` for session and user storage in PostgreSQL.

## Environment Variables

| Variable | Description |
|---|---|
| `AUTH_SECRET` | Secret key for signing session tokens |
| `AUTH_GITHUB_ID` | GitHub OAuth app client ID |
| `AUTH_GITHUB_SECRET` | GitHub OAuth app client secret |
| `AUTH_GOOGLE_ID` | Google OAuth client ID |
| `AUTH_GOOGLE_SECRET` | Google OAuth client secret |

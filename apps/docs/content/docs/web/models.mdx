## Default Model

The default model is **`anthropic/claude-sonnet-4.5`** (Claude Sonnet 4.5).

When the `selectedModel` field is omitted from a chat request, this model is used.

## Model Selection

Models are defined in `packages/ai/src/models.ts` and exported as:

- `chatModels` — Array of all available models
- `modelsByProvider` — Models grouped by provider
- `DEFAULT_CHAT_MODEL` — The default model ID

## Adding New Models

To add a new model, add an entry to the `chatModels` array in `packages/ai/src/models.ts`:

```typescript
{
  id: "provider/model-name",
  name: "Display Name",
  provider: "provider",
  description: "Short description",
}
```

The model will automatically appear in the UI model selector and be available via the chat API.

## Reasoning Models

Models in the `reasoning` category (like Claude 3.7 Sonnet Thinking) use a simplified system prompt without artifact instructions, since extended thinking models work best with concise prompts.
